## üïπÔ∏è Gameplay Overview

You play a university student doing **‚Äúcampus running practice‚Äù**, with one clear goal:  
**Find the correct exit door and escape before the enemy catches you.**

1. **Run & Escape:** Always keep moving ‚Äî time is life.
2. **Path Selection:** At rooms with multiple doors or branches, you must make quick decisions ‚Äî **only one door leads to the correct escape route**.
3. **Obstacle Avoidance:** Avoid environmental obstacles or fallen objects to maintain smooth running.
4. **Stamina Management:** Continuous sprinting drains stamina.
5. **Bullet Time:** Hold the left mouse button to slow down enemies and escape more safely. Sometimes you can even use this feature to show off.

## üöÄ Getting Started (For Developers)

#### 1. Clone the Repository

```bash
git clone https://github.com/new-tonAA/ResidentRun-UE5Game.git
```

#### 2. Engine Setup

Make sure you have Unreal Engine 5.6.1 or above installed.

Ensure Visual Studio 2022 with C++ Desktop Development components is installed.

#### 3. Open & Run

Open ResidentRun-UE5Game.uproject in the project root directory.

Wait for UE to compile C++ code (if required).

In the UE5 editor, open the main map Map_Start, then click Play to start testing.

#### 4. Note

This repository does not include the full source code; many large files are not uploaded.
Therefore, a direct build may not fully reproduce the game.
If you need the complete source code, please contact via email.
